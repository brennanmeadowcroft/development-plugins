# Development Tools

Skills to support the plan-driven feature development workflow with Claude Code.

## Targeted Workflow

These tools streamline the following development workflow:

1. Work with Claude in planning mode to develop a plan
2. Save the plan to a folder within the `plans` folder at the project root
3. Start other sessions to build an entire plan or specific phases for large projects

## Skills

| Skill                      | Description                                                                                                                             |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `/build-feature-from-plan` | Reviews the plan and builds the feature according to the plan. Targets a specific phase if directed, otherwise builds all phases. Designed for platform server projects only. |
