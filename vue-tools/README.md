# Vue Tools

Vue Tools brings expert Vue.js architecture guidance into the development workflow. The plugin provides a specialized agent that advises on component design, state management patterns, and codebase organization — with particular focus on large-scale, multi-team applications.

The agent applies a simplicity-first philosophy: it favors the simplest solution that adequately solves the problem, and only introduces architectural overhead when there is a clear, demonstrable benefit.

## Skills

None

## Agents

| Agent | Description |
|---|---|
| Vue Architect | Reviews Vue component implementations, advises on architecture decisions (component structure, state management, feature organization), and evaluates code against Vue 3 best practices. Runs proactively when the user needs guidance on a Vue architecture decision or requests a component review. |
